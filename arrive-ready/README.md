# ArriveReady ✈️

**Land energized. Capture every moment.**

*A Capture Adventure Now tool.*

---

ArriveReady is a personalized jet lag planner built on circadian science — the same approach trusted by NASA, elite athletes, and F1 teams. Enter your travel details and get a day-by-day plan covering sleep timing, light exposure, hydration, food, and more.

## Features

- **Sleep & Light Schedule** — personalized bedtime, wake time, and light exposure windows
- **Hydration Guide** — water targets, electrolyte guidance, and what to avoid
- **Food & Diet** — what to eat and when, aligned to your circadian phase
- **Daily Tips** — behavioral tips tailored to pre-trip, travel day, and destination
- **Extras** — exercise timing, temperature therapy, stress management, and gear checklist

## Tech Stack

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)

## Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build
```

## Deploying to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **Add New Project** and select this repo
4. Vercel auto-detects Vite — just click **Deploy**
5. Your app is live at `your-project.vercel.app`
